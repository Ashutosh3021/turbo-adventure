## Technologies Used

- Vanilla JavaScript (no frameworks)
- OpenRouter API for AI responses
- Progressive Web App standards
- Service Worker for offline functionality
- LocalStorage for data persistence

## Personalities

The app comes with several pre-configured AI personalities:
- **JARVIS**: Helpful and efficient assistant
- **Data**: Analytical and fact-based personality
- **Yoda**: Wisdom-filled responses in a unique speech pattern
- **C-3PO**: Polite and protocol-focused personality

## Customization

Create your own AI personalities by:
1. Opening the settings panel
2. Navigating to the "New Character" tab
3. Providing a name, system prompt, and color
4. Saving your new character

## API Key

To use the chat functionality, you'll need an API key from [OpenRouter](https://openrouter.ai/):
1. Sign up for an account at OpenRouter
2. Generate an API key
3. Enter the key in the settings panel
